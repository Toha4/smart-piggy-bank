# Smart Piggy Bank (Универсальная копилка)

Веб-приложение для отслеживания прогресса накопления на любую финансовую цель. Первоначальный кейс — накопление на Sony PlayStation 5.

## Описание проекта

Smart Piggy Bank — это веб-приложение, которое помогает отслеживать прогресс накопления на любую финансовую цель. Приложение позволяет:
- Устанавливать цель (название, сумму, дедлайн)
- Пополнять копилку и отслеживать текущий баланс
- Просматривать историю всех операций
- Редактировать и удалять транзакции
- Настраивать внешний вид приложения
- Визуализировать прогресс с изменением цвета прогресс-бара и праздничной анимацией при достижении цели

## Архитектура

Проект состоит из трех основных компонентов:

### Frontend
- **Фреймворк**: React 18 с TypeScript
- **Сборка**: Vite
- **Стили**: Tailwind CSS
- **Дополнительно**: Axios (HTTP), React Router, canvas-confetti (анимация)

### Backend
- **Фреймворк**: FastAPI (Python 3.11+)
- **ORM**: SQLAlchemy 2.0
- **Валидация**: Pydantic 2.0
- **Сервер**: Uvicorn (ASGI)

### База данных
- **СУБД**: SQLite 3
- **Хранение**: Файл в директории проекта

### Инфраструктура
- **Контейнеризация**: Docker + Docker Compose
- **Развертывание**: Все компоненты (фронтенд, бэкенд, БД) разворачиваются локально в контейнерах

## Сущности

### Цель (Goal)
- Название цели (например, "PlayStation 5", "Новый ноутбук")
- Целевая сумма (в рублях)
- Текущий баланс (рассчитываемый автоматически на основе операций)
- Опционально: Изображение цели, срок (дедлайн)

### Операция (Transaction)
- Сумма операции (положительное число — пополнение)
- Дата и время операции (проставляются автоматически при создании)
- Комментарий (необязательный текст, например, "Аванс", "Подарок")

## API Эндпоинты

### Цели
- `GET /goals/goal` - получить данные о текущей цели
- `POST /goals/goal` - создать новую цель
- `PUT /goals/goal/{goal_id}` - обновить цель

### Транзакции
- `GET /transactions/transactions` - получить список транзакций
- `POST /transactions/transactions` - создать новую транзакцию
- `PUT /transactions/transactions/{transaction_id}` - обновить транзакцию
- `DELETE /transactions/transactions/{transaction_id}` - удалить транзакцию

### Настройки
- `GET /settings/theme` - получить настройки темы
- `PUT /settings/theme` - обновить настройки темы

## Установка и запуск

### Локальный запуск

#### Backend
1. Установите зависимости:
```bash
pip install -r backend/requirements.txt
```

2. Запустите приложение:
```bash
cd backend
python -m uvicorn main:app --reload
```

Backend будет доступен по адресу `http://localhost:8000`.

#### Frontend
1. Установите зависимости:
```bash
cd frontend
npm install
```

2. Запустите приложение:
```bash
npm run dev
```

Frontend будет доступен по адресу `http://localhost:5173`.

### Запуск с Docker

1. Убедитесь, что у вас установлен Docker и Docker Compose
2. Выполните команду в корневой директории проекта:
```bash
docker-compose up --build
```

Приложение будет доступно по адресу `http://localhost:5173`, а API по адресу `http://localhost:8000`.
